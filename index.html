YouToo uses the YouTube Data API v3.

Get a free API key (2 min):
1. console.cloud.google.com → Create project
2. APIs & Services → Library → YouTube Data API v3 → Enable
3. Credentials → Create API Key
4. Restrict to your domain (recommended)

Key is saved to your browser's localStorage only — never sent anywhere except Google.
